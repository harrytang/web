# MDX meta data content rules

When generating meta content for MDX files, please adhere to the following character limits and guidelines:

When the user asks to generate page meta content (title, description, image, open graph, twitter card), always follow these character limits:

- meta title: 50-60 characters (sentence case)
- meta description: 110-160 characters
- Keyword Density: 1-2% for the main keyword, and 0.5-1% for secondary keywords.
- open graph title: Different from meta title, focus for social sharing, should not exceed 60 characters (Sentence case).
- open graph description: Different from meta description, focus for social sharing, should not exceed 65 characters.
